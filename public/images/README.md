# Logo Image

Please place your ABRAJ STAY logo image file here.

**File name:** `abraj-stay-logo.png`

**Recommended specifications:**
- Format: PNG (with transparent background preferred)
- Height: 200-300px (will be scaled automatically)
- Width: Auto (maintain aspect ratio)
- File size: Optimized for web (< 100KB)

The logo will be displayed in:
- Navigation bar (header)
- Footer section
- Copyright section

If you don't have the logo file yet, the site will show a broken image icon. Once you add the logo file to this directory, it will automatically appear throughout the website.
